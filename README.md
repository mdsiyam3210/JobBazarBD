## Tech Stack

**Client:** Angular-18, Angular Material, HTML, SCSS, CSS  

**Server:** Springboot, Java 


## Installation
Clone both repos and do the following:-

FRONTEND
```bash
  npm install
  npm audit fix
  ng serve --open
```

BACKEND
-  Install maven dependencies,
-  Run JobListingApplication.java
